What is HTML?

HTML stands for Hyper Text Markup Language, and it is indeed the standard markup language for creating web pages. HTML describes the structure of a web page using a series of elements, and these elements tell the browser how to display the content. For example, to convert text into a paragraph, you would use the <p> element in HTML.

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>My First Heading</h1>
    <p>My first paragraph.</p>

</body>
</html>
  • <!DOCTYPE html> declaration defines that this document is an HTML5 document.
  • <html> element is the root element of an HTML page.
  • <head> element contains meta information about the HTML page.
  • <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).
  • <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • <h1> element defines a large heading.
  • <p> element defines a paragraph.

HTML Attributes

All HTML elements can have attributes. These attributes provide additional information about the elements. They are always specified in the start tag of the element. Typically, attributes come in name/value pairs like: name="value".The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

<a href="https://www.w3schools.com">Visit W3Schools</a>

The tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:

<img src="img_girl.jpg">

HTML Headings and Text Formatting

HTML provides various heading elements (<h1> to <h6>) for defining different levels of headings, as well as text formatting tags such as <strong>, <em>, <u>, and <strike>. 

<h1>Main Heading</h1>
<h2>Subheading</h2>
<p><strong>Bold Text</strong></p>
<p><em>Italicized Text</em></p>
<p><u>Underlined Text</u></p>
<p><strike>Strikethrough Text</strike></p>

HTML Lists

HTML supports both ordered lists (<ol>) and unordered lists (<ul>), which can contain list items (<li>) 

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
</ol>
